本站实现方式

2022-04-14

这篇文章我会大概讲一下本站的实现方式，也算是给后人铺了两块石头
当然，Alan我也算大半个萌新，可能会有讲的不对的地方，请多体谅。

Github Pages

首先，说说托管本站的GitHub Pages

你需要在Github注册一个账号（废话）

例如我的github账号就叫Alanshen2333

然后创建一个新的仓库，名字需叫
yourUserName.github.io

这时候，当你上传一个html之后，你便可以从*https://yourUserName.github.io/*来访问它了

Hexo

显然，让我从头学js开始写站点是不现实的其实就是我菜。

因此，我用到了Hexo来生成我的站点

过程很简单，官方的文档也写得很全面
https://hexo.io/

在简单的用nodejs装完了Hexo后

简单的

hexo s

你应该就能从
http://[::1]:4000/
访问你的网站了
当然它此时就和本站此时一样什么都没有

你可以用
hexo new
来写一篇文章，需使用markdown语法

*（值得一提的是，网站渲染出的markdown与我vscode中markdown插件的预览并不完全一致。）

扯远了

我们回到*_config.yml*

你需要更改文件的末尾

像这样

deploy:
 type: git
 repository: https://github.com/Alanshen2333/alanshen2333.github.io.git
 branch: main

接着使用

hexo g -d

便能将网站同步到GitHub Pages

美化

在完成了基本的内容后，就到了美化的时候了，其实默认的模板挺好看的，但我不知道咋自定义-.-

于是我就用到了Hexo的 themes

我用到的是

https://github.com/luswdev/hexo-theme-clean

按文档装完后替换一下resource里的资源文件就行了
记得注意文件拓展名

好，差不多就写这些罢。
